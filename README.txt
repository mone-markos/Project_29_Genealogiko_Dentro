ΓΕΝΕΑΛΟΓΙΚΟ ΔΕΝΤΡΟ: Δημιουργήθηκε μια εφαρμογή η οποία δημιουργεί ένα γενεαλογικό δέντρο χρήστη και περιλαμβάνει τα παρακάτω.

Α. Γραφικό περιβάλλον για εισαγωγή δεδομένων όπου εμφανίζονται φόρμες για δημιουργία ατόμου:
	1. Όνομα
	2. Επώνυμο
	3. Ημερομηνία γέννησης

καθώς και σχέσεις μεταξύ τους:
	1. Γονείς-Παιδί
	2. Σύζυγοι
	3. Αδέρφια

Β. Δυνατότητα επεξεργασίας και διαγραφής του ατόμου

Γ. Αποθήκευση δεδομένων με δυνατότητα αποθήκευσης τους τοπικά και εξασφάλιση παραμονής της πληροφορίας και μετά από επανεκκίνηση της εφαρμογής.

Δ. Απεικόνιση του γενεαλογικού δέντρου σε μορφή γράφου όπου κάθε άτομο απεικονίζεται ως κόμβος και η σχέση μεταξύ τους με γραμμή χρωματισμένη ανάλογα με την σχέση τους.

Ε. Κάνοντας κλικ σε κόμβο, ανοίγει pop up παράθυρο με τις πληροφορίες του ατόμου.

ΣΤ. Αναζήτηση ατόμων στην εφαρμογή χρησιμοποιώντας το ID, Όνομα, Επώνυμο.
	
Χρησιμοποιήθηκαν:
1. Γλώσσα προγραμματισμού: Python
2. Γραφικές βιβλιοθήκες: tkinter, matplotlib, networkx
3. Αποθήκευση δεδομένων: SQLite
4. Αναζήτηση:Python, SQLite
