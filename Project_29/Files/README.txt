===============================
Γενεαλογικό Δέντρο - Project 29
===============================

Περιγραφή:
Αυτή η εφαρμογή επιτρέπει τη δημιουργία και διαχείριση ενός γενεαλογικού δέντρου μέσω γραφικού περιβάλλοντος χρήστη (GUI).
Ο χρήστης μπορεί να προσθέτει άτομα, να δημιουργεί σχέσεις, να αναζητά απογόνους και να προβάλλει το δέντρο οπτικά.

Εκκίνηση:
- Τρέξτε το αρχείο gui.py με Python.
- Απαιτείται Python 3.10+ και οι βιβλιοθήκες: tkinter, sqlite3, networkx, matplotlib.

Δομή Αρχείων:
- gui.py: Γραφικό περιβάλλον και διαχείριση λειτουργιών
- Database.py: Σύνδεση και λειτουργίες βάσης δεδομένων
- person.py: Ορισμός της κλάσης Person
- search_utility.py: Λογική αναζήτησης απογόνων
- graph.py: Οπτική αναπαράσταση του γενεαλογικού δέντρου

Τεχνολογίες:
- Python, Tkinter, SQLite
- NetworkX και Matplotlib για το γράφημα

Χρήση:
1. Προσθέστε άτομα με το ονοματεπώνυμο και την ημερομηνία γέννησης.
2. Καταχωρήστε σχέσεις μεταξύ ατόμων (γονιός, παιδί, σύζυγος, αδελφός).
3. Χρησιμοποιήστε τα ID των ατόμων για να προσδιορίσετε τις σχέσεις.
4. Αναζητήστε απογόνους ενός ατόμου με βάση το ID του.
5. Δείτε την οπτικοποίηση του γενεαλογικού δέντρου με πλήρη εικόνα.

Σημειώσεις:
- Χρήση από τον «τέλειο χρήστη»: σωστά ID, ημερομηνίες, χωρίς ορθογραφικά λάθη.
