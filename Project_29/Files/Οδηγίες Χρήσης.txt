*Η εφαρμογή έχει δημιουργηθεί να χρησιμοποιείται από τον τέλειο χρήστη. Αυτό σημαίνει πως ο χρήστης εισάγει σωστά τα στοιχεία του, χωρίς ορθογραφικά λάθη και την ημερομηνία σε σωστή μορφή. Συγκρατεί με ακρίβεια όλα τα στοιχεία που έχει εισάγει ώστε να μπορεί να τα χρησιμοποιήσει στην δημιουργία των σχέσεων καθώς και στην αναζήτηση.

Οδηγίες Χρήσης Εφαρμογής Γενεαλογικού Δέντρου
	Η εφαρμογή έχει σχεδιαστεί για χρήση από τον «τέλειο χρήστη». Αυτό σημαίνει ότι:

	Ο χρήστης εισάγει σωστά τα στοιχεία.

	Η ημερομηνία καταχωρείται.

Ο χρήστης θυμάται τα ID των ατόμων που έχει εισάγει ώστε να τα χρησιμοποιεί για σχέσεις ή αναζητήσεις.

Εκκίνηση της Εφαρμογής
	Εκκινήστε την εφαρμογή μέσω του αρχείου gui.py

	Θα εμφανιστεί γραφικό περιβάλλον με κουμπιά και επιλογές.

	Επιλέξτε την αντίστοιχη λειτουργία (Προσθήκη, Διαγραφή, Σχέση, Αναζήτηση κ.λπ.).

Εισαγωγή Στοιχείων
	Πατήστε «Προσθήκη Ατόμου».

Συμπληρώστε:

	Όνομα

	Επώνυμο

	Ημερομηνία γέννησης

	Πατήστε «Αποθήκευση».

Θα εμφανιστεί μήνυμα με το ID που δημιουργήθηκε για το άτομο.

Σημείωση: Κρατήστε το ID για μελλοντική χρήση σε σχέσεις ή αναζήτηση.

Δημιουργία Σχέσης
	Πατήστε «Προσθήκη Σχέσης».

	Εισάγετε:

	ID Ατόμου 1

	ID Ατόμου 2

	Τύπος Σχέσης (γονιός, παιδί, σύζυγος, αδελφός)

	Πατήστε «Αποθήκευση».

Η σχέση θα καταχωρηθεί αν και τα δύο ID είναι έγκυρα.

Επεξεργασία και Διαγραφή
	Επιλέξτε «Επεξεργασία Ατόμου» για να αλλάξετε τα στοιχεία του.

	Επιλέξτε «Διαγραφή Ατόμου» για να αφαιρέσετε κάποιο άτομο. Όλες οι σχετικές σχέσεις διαγράφονται αυτόματα.

	Επιλέξτε «Επεξεργασία/Διαγραφή Σχέσης» για αλλαγή ή αφαίρεση σύνδεσης μεταξύ ατόμων.

Αναζήτηση Απογόνων
	Πατήστε «Αναζήτηση Απογόνων».

	Εισάγετε το ID του ατόμου που θέλετε να ελέγξετε.

	Πατήστε «Αναζήτηση».

Θα εμφανιστεί λίστα με όλους τους απογόνους του, ακολουθώντας τη γραμμή σχέσεων τύπου γονιός → παιδί.

Οπτικοποίηση Δέντρου
	Πατήστε «Προβολή Δέντρου».

	Θα εμφανιστεί το γενεαλογικό γράφημα:

	Οι κόμβοι είναι τα άτομα.

	Οι γραμμές δείχνουν τις σχέσεις τους.

	Κάνοντας κλικ σε κόμβο, εμφανίζονται πληροφορίες για το άτομο.
